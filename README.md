## 📋 الجداول المعروضة

| الجدول | الوصف |
|--------|-------|
| `visitors` | جدول الزوار الرئيسي مع بيانات الهوية والمركبة والتأمين |
| `visitor_history` | سجل زيارات الزوار وتتبع الخطوات |
| `visitor_messages` | رسائل الزوار مع الإدارة |
| `insurance_companies` | شركات التأمين |
| `insurance_offers` | عروض التأمين |
| `offer_features` | الميزات الإضافية للعروض |
| `offer_expenses` | الرسوم الإضافية |
| `public_settings` | الإعدادات العامة |
| `socket_sessions` | جلسات Socket |
| `activity_log` | سجل النشاط |

## ✨ المميزات

### 📊 لوحة التحكم الرئيسية
- إحصائيات عامة (الزوار، التأمينات، الإيرادات)
- رسوم بيانية للزيارات والتأمينات
- توزيع أنواع التأمين
- حالة الدفع
- النشاط الأخير

### 👥 إدارة الزوار
- قائمة الزوار مع البحث والتصفية
- تتبع الخطوات
- حالة الدفع
- التصدير

### 📑 إدارة التأمينات
- قائمة التأمينات الكاملة
- البحث والتصفية حسب النوع
- تفاصيل التأمين والقيمة

### 🏢 شركات التأمين
- إدارة شركات التأمين
- العروض لكل شركة
- التفعيل/الإيقاف

### 💬 الرسائل
- نظام رسائل متكامل
- محادثات مع الزوار
- إشعارات غير مقروءة

### 📈 التحليلات
- اتجاه الزيارات
- الإيرادات الشهرية
- قانون التحويل
- توزيع استخدام المركبات
- أداء شركات التأمين

### ⚙️ الإعدادات
- الإعدادات العامة
- إعدادات الأمان
- الدول المحظورة
- البنوك المحظورة
- الإشعارات
- إعدادات البريد

## 🛠️ التقنيات المستخدمة

- **React 19** - إطار العمل
- **Vite** - أداة البناء
- **Tailwind CSS 4** - التصميم
- **Recharts** - الرسوم البيانية
- **Lucide React** - الأيقونات
- **React Router** - التوجيه

## 🚀 تشغيل المشروع

```bash
# تثبيت التبعيات
cd becare-dashboard
npm install

# تشغيل في وضع التطوير
npm run dev

# بناء للإنتاج
npm run build
```
